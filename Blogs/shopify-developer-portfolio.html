Featured Projects

ZARR by Jazz — zarr.com.pk

Type: Pakistan's digital marketplace | Completed: November 2025

ZARR is Jazz's digital commerce platform, built to serve Jazz's 70M+ subscriber base across Pakistan. This was the most technically complex project in Ibrar's portfolio: a custom marketplace architecture built on Shopify Liquid combined with React components, handling multi-vendor product listings, dynamic pricing, subscriber authentication via Jazz APIs, and a mobile-first UX designed for Pakistan's predominantly mobile internet users.

The technical stack included Shopify as the commerce backend, React components embedded in Liquid templates for dynamic sections, custom Order Management APIs for multi-vendor fulfillment routing, and a performance-optimized frontend targeting 90+ PageSpeed on mobile. ZARR represents the intersection of enterprise-scale commerce requirements and Shopify's flexibility as a backend platform — a project that required genuine senior-level Shopify and React expertise to execute.

  • Custom marketplace architecture on Shopify Liquid + React
  • 70M+ potential user base (Jazz subscribers)
  • Multi-vendor product and order management
  • Mobile-first UX optimized for Pakistan market
  • Launched: November 2025

Chinyere — chinyere.pk

Type: Pakistan's leading fashion brand | Result: 180% organic traffic increase

Chinyere is one of Pakistan's most recognized women's fashion brands, with hundreds of SKUs across kurtas, formal wear, and seasonal collections. Ibrar's work on Chinyere encompassed two major deliverables: a custom Shopify theme build tailored to the brand's visual identity and customer journey, and a comprehensive SEO overhaul that delivered a 180% increase in organic traffic.

The technical work included: rebuilding product page templates with correct structured data (Product, BreadcrumbList, Organization schemas), implementing collection page SEO with custom meta templates, fixing crawlability issues identified through Search Console audits, building custom Liquid sections for seasonal promotional content, implementing a size guide system with Pakistani sizing context, and integrating Instagram Shopping for social commerce. The 180% traffic increase was achieved over 8 months of consistent SEO work combined with a technically sound site structure.

  • Custom Shopify theme development aligned with brand identity
  • 180% organic traffic increase (documented via Google Analytics)
  • Full structured data implementation (Product + Collection schemas)
  • Size guide system with Pakistani sizing conventions
  • Instagram Shopping catalog integration

She Matters — shemattersofficial.com

Type: React.js women's platform | Stack: React.js + custom components

She Matters is a women-focused platform built entirely in React.js. This project was a frontend-first build: Ibrar designed and developed custom React components from scratch — navigation, hero sections, content modules, and interactive elements — achieving a modern, performant web experience without relying on any pre-built component library. The result is a site with a distinctive visual identity and smooth interactions that a template-based build could never deliver.

SEO on a React SPA requires deliberate implementation: server-side rendering for all SEO-critical pages, structured data via JSON-LD injected server-side, proper canonical tags, and a sitemap generated from the application's route structure. Ibrar implemented all of these, and the site achieves strong organic visibility for its target keywords in the Pakistani market.

  • Full React.js frontend built from scratch
  • Custom component library for the brand
  • SEO-optimized with server-side rendering
  • Structured data and schema markup
  • High-performance score on mobile and desktop

Minnie Minors — Kids Fashion Shopify Store

Type: Kids fashion e-commerce | Features: Custom filters, size guides, wishlist

Minnie Minors is a kids' fashion brand requiring age-specific size guides, gender-based collection filtering, and a wishlist feature for gift-buying use cases. Ibrar built custom Liquid templates for the collection pages with multi-attribute filtering (age range, gender, occasion, size), an age-appropriate size guide system that maps sizes to age ranges rather than standard measurements, and a wishlist feature using localStorage-based state management for guest users and customer account metafields for logged-in users.

The kids fashion market has specific UX considerations: parents are often buying for children who aren't present, so size guidance needs to be especially clear, and gift occasions (birthdays, Eid) drive significant seasonal purchase intent. Ibrar's work addressed these nuances with targeted content sections and a gift-friendly checkout experience.

  • Custom collection filters (age, gender, occasion, size)
  • Age-range size guides for kids' clothing
  • Wishlist system (guest + logged-in users)
  • Gift-purchase optimized checkout flow
  • Seasonal campaign landing page templates

MedVoyage Global — medvoyageglobal.com

Type: React.js medical tourism platform | Stack: React + healthcare schema + multilingual SEO

MedVoyage Global is a medical tourism platform connecting international patients with hospitals and healthcare providers. This React.js project required specialized healthcare-specific SEO implementation: MedicalOrganization and Physician structured data schemas, multilingual content for multiple target markets, and local SEO signals for the healthcare providers featured on the platform. The challenge was building a platform that performs well in organic search across multiple languages and countries simultaneously.

Ibrar built custom React components for hospital and physician profile pages, implemented multilingual routing with hreflang tags, added healthcare-specific structured data for treatment types, provider credentials, and geographic service areas, and ensured HIPAA-appropriate content handling (no patient data collected via the platform). The result is a medical tourism site that ranks organically in its target markets and converts international healthcare inquiries.

  • React.js frontend with custom medical profile components
  • Healthcare structured data (MedicalOrganization, Physician schemas)
  • Multilingual SEO with hreflang implementation
  • International market targeting across multiple regions
  • HIPAA-appropriate content architecture

eJobber — ejobber.co.uk

Type: Laravel + React job platform | Market: United Kingdom

eJobber is a UK job platform built on a Laravel backend with a React frontend — demonstrating Ibrar's capabilities beyond Shopify. The platform required user authentication, job listing management with geographic filtering, applicant tracking, and email notification workflows. The React frontend was built with performance and accessibility in mind for the UK market, where accessibility compliance (WCAG 2.1) is increasingly important for web platforms.

The UK job market is competitive, and eJobber's SEO required structured data for job listings (JobPosting schema), geographic targeting for UK job searches, and content strategy aligned with UK employment search patterns. Ibrar implemented the JobPosting schema across all listings, configured regional targeting in Google Search Console, and built an SEO-friendly URL structure for job category and location pages.

  • Laravel + React full-stack development
  • Job listing management with location-based filtering
  • JobPosting structured data for all listings
  • UK market SEO targeting
  • WCAG 2.1 accessibility compliance
