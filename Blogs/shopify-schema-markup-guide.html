Shopify Schema Markup Guide: Complete Structured Data Implementation with Liquid Code

How to add Schema.org structured data to your Shopify store — Product, BreadcrumbList, FAQ, and Article schemas with real Liquid code examples, by Ibrar Zahoor.

April 11, 2026 14 min read Ibrar Zahoor

Schema markup is the bridge between your Shopify store's content and Google's understanding of it. Properly implemented structured data unlocks rich results — star ratings, price information, FAQ accordions, and breadcrumbs — directly in search results. This increases click-through rates by 20-30% and is a key factor in AI Overview inclusion. This guide by Ibrar Zahoor covers every schema type relevant to Shopify stores, with working Liquid code you can implement today.

What is Schema Markup and Why Does it Matter for Shopify?

Schema markup is structured data — a standardized vocabulary (Schema.org) that describes your content to search engines in a machine-readable way. Google uses it to:

  • Display star ratings and price ranges in product search results (rich snippets)
  • Show FAQ accordions directly in SERPs (FAQPage)
  • Display breadcrumb navigation in search results
  • Populate Google Shopping with accurate product data
  • Include content in AI Overviews and featured snippets

All Shopify schema should be implemented as JSON-LD — Google's recommended format — inside a <script type="application/ld+json"> tag.

Shopify Product Schema: The Most Important Implementation

Product schema enables rich results for price, availability, ratings, and images in Google Shopping and organic search. Here is a complete Product schema implementation in Liquid:

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "{{ product.title | escape }}",
  "description": "{{ product.description | strip_html | escape | truncate: 500 }}",
  "url": "{{ shop.url }}{{ product.url }}",
  "image": "{{ product.featured_image | img_url: 'master' }}",
  "sku": "{{ product.selected_or_first_available_variant.sku | escape }}",
  "brand": {
    "@type": "Brand",
    "name": "{{ product.vendor | escape }}"
  },
  "offers": {
    "@type": "Offer",
    "price": "{{ product.price | money_without_currency }}",
    "priceCurrency": "{{ cart.currency.iso_code }}",
    "availability": "{% if product.available %}https://schema.org/InStock{% else %}https://schema.org/OutOfStock{% endif %}",
    "url": "{{ shop.url }}{{ product.url }}"
  }
}
</script>
Ibrar Zahoor's Note: Place this snippet in your product.liquid or main-product.liquid section file, not in the theme layout. Schema on non-product pages causes errors in Google Search Console.

Adding AggregateRating to Product Schema

If you use a reviews app like Judge.me, Yotpo, or Okendo, check if the app already injects AggregateRating schema. If not, add it to your Product JSON-LD:

"aggregateRating": {
  "@type": "AggregateRating",
  "ratingValue": "{{ product.metafields.reviews.rating.value }}",
  "reviewCount": "{{ product.metafields.reviews.rating_count.value }}"
}

BreadcrumbList Schema for Shopify

Breadcrumb schema shows your navigation path in Google search results, improving CTR and helping users understand your site structure before clicking.

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "Home",
      "item": "{{ shop.url }}"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "{{ collection.title | escape }}",
      "item": "{{ shop.url }}{{ collection.url }}"
    },
    {
      "@type": "ListItem",
      "position": 3,
      "name": "{{ product.title | escape }}"
    }
  ]
}
</script>

FAQPage Schema for Shopify

FAQPage schema can generate FAQ accordions in Google search results — a highly visible rich result that pushes competitors down the page. Use it on collection pages, product pages (for product FAQs), and standalone FAQ pages.

Schema TypePage TypeRich Result Appearance
ProductProduct pagesStar ratings, price, availability
BreadcrumbListAll pagesBreadcrumb path in SERP
FAQPageFAQ, product, collection pagesExpandable Q&A accordion in SERP
ArticleBlog post pagesArticle metadata, date, author
OrganizationHomepage, about pageBusiness info in Knowledge Panel
SiteNavigationElementAll pages (header)Sitelinks

FAQPage Liquid Implementation with Metafields

Store FAQ data in Shopify metafields (a JSON metafield with an array of {question, answer} objects) and generate both the visible HTML and JSON-LD dynamically:

{%- assign faqs = product.metafields.custom.faqs.value -%}
{%- if faqs -%}
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {%- for faq in faqs -%}
    {
      "@type": "Question",
      "name": {{ faq.question | json }},
      "acceptedAnswer": {
        "@type": "Answer",
        "text": {{ faq.answer | json }}
      }
    }{% unless forloop.last %},{% endunless %}
    {%- endfor -%}
  ]
}
</script>
{%- endif -%}

Article Schema for Shopify Blog Posts

Article schema helps Google understand your blog content, attribute authorship, and potentially include articles in Discover and AI Overviews. Ibrar Zahoor includes Article schema on all blog posts he builds.

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "{{ article.title | escape }}",
  "description": "{{ article.excerpt | strip_html | escape | truncate: 155 }}",
  "datePublished": "{{ article.published_at | date: '%Y-%m-%d' }}",
  "dateModified": "{{ article.updated_at | date: '%Y-%m-%d' }}",
  "author": {
    "@type": "Person",
    "name": "{{ article.author }}"
  },
  "publisher": {
    "@type": "Organization",
    "name": "{{ shop.name | escape }}"
  },
  "image": "{{ article.image | img_url: 'master' }}"
}
</script>

Organization Schema: Establish Your Brand Identity

Organization schema on your homepage and about page helps Google build a Knowledge Panel for your brand and populate business information in branded search results.

Key Properties to Include: name, url, logo, contactPoint (phone/email), sameAs (links to your social profiles — LinkedIn, Instagram, Facebook). The sameAs array is especially important for entity disambiguation and Knowledge Panel population.

Common Shopify Schema Errors to Fix

  • Missing required fields: Google requires at minimum name, description, image, and offers for Product schema.
  • Price format mismatch: Ensure price values are numeric (no currency symbols inside the price field; use priceCurrency separately).
  • Duplicate schema: Two competing Product schema blocks (often from app + theme) cause errors. Remove duplicates.
  • Schema on wrong page type: FAQPage schema on every page without FAQ content triggers GSC warnings.
  • Outdated availability values: Use the full Schema.org URLs: https://schema.org/InStock, not just "InStock".

Testing and Monitoring Shopify Schema

Schema Testing Tools

  • Google Rich Results Test: search.google.com/test/rich-results — test any URL or code snippet
  • Schema.org Validator: validator.schema.org — validate JSON-LD syntax
  • Google Search Console: Enhancements → Products/FAQ/Breadcrumbs — monitor coverage and errors
  • Screaming Frog SEO Spider: Bulk audit schema across all Shopify pages
